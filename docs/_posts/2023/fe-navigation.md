## 其他

- **patch-package** 给 npm 包打补丁，快速修复问题。 在 node_modules 中修改package 源码后，可以通过 patch-package 生成相应的补丁包，避免重新安装以来后丢失修改内容，并可以在npm install 后自动应用相应的补丁。
